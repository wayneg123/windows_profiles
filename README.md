# Windows profiles
These are my windows profiles

**install.bat still in testing**

If you still want to use it, you can follow instroctions below.
# Simple Use:
`win+r` to run CMD.
```
cd %userprofile%
git clone https://github.com/wayneg123/windows_profiles.git
cd windows_profiles
install.bat
```
